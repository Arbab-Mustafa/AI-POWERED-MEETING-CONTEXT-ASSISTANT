INFORMATION ARCHITECTURE
================================================================================

You need to set up 3 types of variables:

TYPE 1: CREDENTIALS (Where To Get)
├─ PostgreSQL password → You create during pg installation
├─ Google OAuth → Google Cloud Console
├─ Gmail app password → Gmail account settings
├─ Telegram token → @BotFather on Telegram
└─ Ollama URL → Local installation

TYPE 2: CONFIGURATION (How To Set)
├─ API endpoints → You decide (localhost for dev)
├─ Application name/version → You decide
├─ Environment type → You decide (dev/prod)
└─ Debug mode → You decide

TYPE 3: SECURITY (How To Handle)
├─ Never commit to GitHub
├─ Never share with others
├─ Keep in password manager
├─ Regenerate if compromised
└─ Rotate keys periodically
